# Project Plan: CoastVision

## Phases

1. **Data Collection & Annotation**
   - Gather public datasets and videos.
   - Annotate for human presence and drowning behavior.

2. **Model Development**
   - Human detection using YOLOv5/v8.
   - Behavior analysis using CNN+LSTM.

3. **Backend & Alert System**
   - Real-time video processing.
   - Alert delivery to lifeguard app.

4. **Frontend & Dashboard**
   - Lifeguard mobile/web app.
   - Heatmap and analytics dashboard.
